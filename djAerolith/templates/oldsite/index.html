Aerolith


Hi, my name is Cesar and Aerolith is my word study tool. Among its features are multiplayer, and a cycle mode that allows you to go thru lists repeatedly until you get all the answers!

Updates

If you have an older version of Aerolith, please upgrade using the links below. I'll try to make it so that the client doesn't have to be updated that often (i.e. make the server backwards compatible with older versions of the client, if the user does not wish to upgrade).

Current version: 0.4.1.1
Changes:
- Added a new display mode under Preferences - boxes around tiles.
- Added a new display mode -- no tiles at all (words are displayed in rectangles instead)
- Toggle font button for Word Rectangle mode.
- Clicking on words to shuffle them no longer makes Guess box lose focus.
- Enlarged guess box.
- Insert only meaningful guesses into Guesses box.


Coming soon:
- ODS and Collins
- Marathon mode
Aerolith for Windows Run the exe to install Aerolith. Uninstall the previous version of Aerolith, by going to the Start Menu, Programs, select Aerolith, and click the Uninstall Aerolith option.
Aerolith for Mac - Universal This is a self contained dmg file. It will not work on OSX version 10.3.8 and lower.
